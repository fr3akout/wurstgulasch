Wurstgulasch
============

What is Wurstgulasch?
---------------------
Wurstgulasch aims to be a decentral (kind of) social blog application, i.e. a social "network" minus the annoying parts. 


Dependencies
------------
wurstgulasch depends on:
  * werkzeug
  * jinja2
  * PIL
  * sqlalchemy
  * wtforms
  * beaker
